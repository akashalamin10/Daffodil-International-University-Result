## 🧩 Features

✅ **No Login Required** – Just enter your Student ID  
✅ **Instant Results** – All semester grades, total credits, and CGPA  
✅ **Fast & Lightweight** – Caches data locally for quicker access  
✅ **Responsive Design** – Works perfectly on mobile and desktop  
✅ **Download Transcript** – High-quality PDF export  
✅ **Beautiful Interface** – Clean and intuitive user experience
